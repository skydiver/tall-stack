## Run Locally

Clone the project

```bash
  git clone https://github.com/skydiver/tall-stack.git
```

Go to the project directory

```bash
  cd tall-stack
```

Install dependencies

```bash
  npm install
  composer install
```

Start the server

```bash
  npm run dev
```
